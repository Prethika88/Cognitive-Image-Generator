## Cognitive-Image-Generator

- The Cognitive Image Generator is a full-stack AI application that generates images from text prompts using an AI model.

- Users can enter a text description, and the system generates an image, stores it in Cloudinary, and saves the image metadata in MongoDB.


## Technologies Used

**Frontend**: React, TypeScript, Vite, Tailwind CSS 

**Backend**: Node.js, Express.js, TypeScript

**Database**: MongoDB, Cloudinary 

**AI Service**: Hugging Face Image Generation API


## Features

- Generate images from text prompts

- AI-powered image generation (via Hugging Face API)

- Images stored securely in Cloudinary
  
- Image metadata stored in MongoDB


## Installation

**1.Clone the Repository**

git clone https://github.com/Prethika88/Cognitive-Image-Generator.git

cd Cognitive-Image-Generator

**2.Frontend Setup**

cd frontend

npm install

npm run dev

**3.Backend Setup**

cd backend

npm install

npm start

**4.Setup Environment Variables**

Create a .env file in the server directory and configure the required environment variables.

**5.Run the application**

Frontend: http://localhost:5173

Backend: http://localhost:8080
